# Dashboard Overview

OneClick's dashboard is built for traders who need lightning-fast access to market opportunities. The interface combines Twitter monitoring with token management in a seamless, resizable layout.

### Dual-Panel Design

## Top Navigation Bar

The top bar provides quick access to essential functions:

### Left Side
- **OneClick Logo**: Return to dashboard home
- **Panel Controls**: Swap panel positions, collapse/expand

### Right Side  
- **My Feed**: Configure Twitter subscriptions and filters
- **Notifications**: Real-time alerts and updates  
- **Settings**: Access wallet, launch, and UI configuration
- **Profile**: Account management and logout

## Key Features

### Real-Time Updates
<Card icon="refresh">
All data updates live without page refreshes. New tweets, launch status changes, and balance updates appear instantly.
</Card>

### Keyboard Shortcuts
<Card icon="keyboard">
Navigate faster with built-in shortcuts:
- `Space`: Scroll to top of active panel
- `N`: Open new launch modal
- `F`: Open My Feed
- `S`: Open Settings
</Card>

### Smart Notifications
<Card icon="bell">
Configurable alerts for:
- New tweets from tracked accounts
- Launch status changes
- Price movements
- Balance updates
</Card>

## Mobile Experience

### Adaptive Interface
- **Single panel view** with tab switching
- **Swipe gestures** for navigation
