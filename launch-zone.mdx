# Launch Zone

The Launch Zone is OneClick's mission control center for token launches. Monitor active deployments, manage positions, execute sells, and track performance all from one powerful interface.

<Card icon="coins" title="What is Launch Zone?">
Launch Zone combines launch queue management, position tracking, and automated selling into a unified workspace. From initial deployment to final exit, manage your entire token lifecycle here.
</Card>

## Queue Management

### Launch Queue

The launch queue shows all pending deployments:

<Steps>
  <Step title="Queue Position">
    **Your place in line** with estimated deployment time based on current network conditions.
  </Step>
  <Step title="Priority Options">
    **Boost deployment speed**:
    - Higher priority fees
    - Express queue access
    - Rush processing (premium feature)
  </Step>
  <Step title="Queue Actions">
    **Manage pending launches**:
    - Cancel queued deployments
    - Modify launch parameters
    - Reorder queue priority
  </Step>
</Steps>

### Network Status

Real-time Solana network information:

- **TPS (Transactions Per Second)**: Current network throughput
- **Slot Height**: Latest confirmed block
- **Fee Recommendations**: Optimal priority fees for fast confirmation
- **Congestion Level**: Network load and expected delays

## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion icon="clock" title="Slow Launches">
    **If deployments are taking too long**:
    - Check Solana network status
    - Increase priority fee settings
    - Consider using express queue
    - Retry during off-peak hours
  </Accordion>
  
  <Accordion icon="x-circle" title="Failed Launches">
    **If launches are failing**:
    - Verify wallet has sufficient SOL for fees
    - Check network congestion levels
    - Review token parameters for errors
    - Contact support with transaction hashes
  </Accordion>
  
  <Accordion icon="refresh" title="Stale Data">
    **If prices/balances aren't updating**:
    - Click the refresh button
    - Check internet connection
    - Clear browser cache
    - Verify RPC endpoint connectivity
  </Accordion>
</AccordionGroup>
