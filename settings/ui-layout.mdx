Optimize OneClick's interface for your trading style with customizable panels, spacing, notifications, and visual preferences. Create the perfect workspace for maximum efficiency.

</Card>

## Notification Position

### Position Options

<Tabs>
  <TabItem value="top-center" label="Top Center">
    **Top center notifications**:
    
    **Characteristics**:
    - Most visible position
    - Non-intrusive to content
    - Standard web pattern
    - Good for all screen sizes
    
    **Best for**: General notifications and alerts
  </TabItem>
  
  <TabItem value="top-right" label="Top Right">
    **Top right notifications**:
    
    **Characteristics**:
    - Traditional desktop position
    - Doesn't block main content
    - Stackable notifications
    - Easy to dismiss
    
    **Best for**: Status updates and confirmations
  </TabItem>
  
  <TabItem value="bottom-center" label="Bottom Center">
    **Bottom center notifications**:
    
    **Characteristics**:
    - Mobile-friendly position
    - Less disruptive scrolling
    - Action-oriented placement
    - Thumb-accessible on mobile
    
    **Best for**: Action prompts and mobile use
  </TabItem>
  
  <TabItem value="bottom-right" label="Bottom Right">
    **Bottom right notifications**:
    
    **Characteristics**:
    - Unobtrusive positioning
    - Traditional chat placement
    - Good for status messages
    - Stackable design
    
    **Best for**: Background updates and logs
  </TabItem>
</Tabs>
