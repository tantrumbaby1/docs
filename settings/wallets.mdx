# Wallet Settings

Manage your Solana wallets for secure trading and token launching. OneClick supports multiple wallets, allowing you to separate trading funds, manage risk, and organize your crypto operations.

<Card icon="wallet" title="Wallet Management">
Access wallet settings through **Settings > Wallets** to generate new wallets, import existing ones, set primary wallets, and monitor balances across all your accounts.
</Card>

## Wallet Operations

### Generate New Wallets

<Steps>
  <Step title="Create Wallet">
    **Click "Generate New Wallet"** to create a fresh Solana wallet:
    - Cryptographically secure key generation
    - Automatic backup phrase generation
    - Instant integration with OneClick
    - Zero manual configuration required
  </Step>
  <Step title="Secure Your Keys">
    **Immediately backup your seed phrase**:
    - Write down all 12 words in order
    - Store in multiple secure locations
    - Never share or store digitally
    - Test recovery before using
  </Step>
  <Step title="Fund Your Wallet">
    **Add SOL for trading**:
    - Send SOL from exchanges or other wallets
    - Start with small amounts for testing
    - Keep extra SOL for transaction fees
    - Monitor balance before launching
  </Step>
</Steps>

### Import Existing Wallets

<AccordionGroup>
  <Accordion icon="key" title="Private Key Import">
    **Import wallets using private keys**:
    - Paste your base58-encoded private key
    - Wallet automatically appears in list
    - Existing balances and tokens load
    - **Security Note**: Only import on trusted devices
  </Accordion>
  
  <Accordion icon="file-text" title="Seed Phrase Import">
    **Restore wallets from backup phrases**:
    - Enter 12 or 24-word seed phrases
    - Automatic derivation path detection
    - Supports standard BIP44 wallets
    - Compatible with Phantom, Solflare, and others
  </Accordion>
  
  <Accordion icon="usb" title="Hardware Wallet Support">
    **Connect hardware wallets** (Coming Soon):
    - Ledger Nano S/X integration
    - Trezor Model T support
    - Enhanced security for large amounts
    - Transaction signing on device
  </Accordion>
</AccordionGroup>

## Security Features

### Private Key Protection

<Warning>
**Critical Security**: OneClick encrypts and stores private keys locally. Never share keys or use untrusted devices for wallet operations.
</Warning>

## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion icon="wifi" title="Connection Problems">
    **If wallets won't connect**:
    - Check internet connection
    - Verify RPC endpoint status
    - Try different browser
    - Clear browser cache
    - Disable browser extensions
  </Accordion>
  
  <Accordion icon="refresh" title="Balance Issues">
    **If balances don't update**:
    - Manual refresh button
    - Check network status
    - Verify wallet addresses
    - Contact support with wallet address
  </Accordion>
  
  <Accordion icon="key" title="Import Failures">
    **If wallet import fails**:
    - Verify private key format
    - Check seed phrase word count
    - Ensure no extra spaces
    - Try different derivation paths
  </Accordion>
</AccordionGroup>

### Recovery Procedures

<Steps>
  <Step title="Document Everything">
    **Before problems occur**:
    - Write down all wallet addresses
    - Backup all seed phrases securely
    - Note custom labels and purposes
    - Record important transaction hashes
  </Step>
  <Step title="Emergency Access">
    **If locked out of OneClick**:
    - Import wallets into Phantom/Solflare
    - Access funds through alternative interfaces
    - Transfer to secure locations
    - Contact support for account recovery
  </Step>
</Steps>
