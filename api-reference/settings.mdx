# Settings API

The Settings API provides endpoints for managing user preferences, wallet configurations, launch settings, and platform customizations within OneClick.

<Card icon="settings" title="Configuration Management">
**Complete settings management** through RESTful endpoints:
- User preferences and defaults
- Wallet management and configuration
- Launch settings and automation
- UI customization and layout preferences
</Card>

## Base URL

```
https://api.oneclick.com/api/settings
```

All endpoints require authentication via Bearer token in the Authorization header.

<Card icon="edit" title="PUT /user">
**Update user preferences and settings**
</Card>

**Request Body:**
```json
{
  "default_sol_amount": 0.2,
  "priority_fee": 7500,
  "auto_launch": true,
  "max_launches_per_day": 15,
  "slippage_tolerance": 0.03,
```

```json
{
```

## Export & Backup

### Export Settings

<Card icon="download" title="GET /export">
**Export all user settings as JSON**
</Card>

**Query Parameters:**
- `format` - Export format: `json`, `yaml`
- `include` - Sections to include: `all`, `user`, `presets`, `wallets`, `ui`

**Response:**
```json
{
  "success": true,
  "data": {
    "export_id": "export_uuid",
    "created_at": "2024-01-20T15:45:00Z",
    "download_url": "https://api.oneclick.com/downloads/export_uuid.json",
    "expires_at": "2024-01-21T15:45:00Z",
    "size_bytes": 15420
  }
}
```

### Import Settings

<Card icon="upload" title="POST /import">
**Import settings from exported JSON**
</Card>

**Request Body:**
```json
{
  "settings": {
    "user": { /* user settings */ },
    "presets": { /* preset configurations */ },
    "ui_layout": { /* UI preferences */ }
  },
  "overwrite": false,
  "backup_existing": true
}
```

## SDK Examples

### JavaScript/TypeScript

```typescript
import { OneClickAPI } from '@oneclick/sdk';

const api = new OneClickAPI({ apiKey: 'your-api-key' });

// Get user settings
const settings = await api.settings.getUser();

// Update preferences
await api.settings.updateUser({
  defaultSolAmount: 0.2,
  priorityFee: 7500,
  autoLaunch: true
});

// Update presets
await api.settings.updateLiquidityPresets([0.01, 0.1, 0.5, 2.0]);

// Configure notifications
await api.settings.updateNotifications({
  channels: {
    email: { enabled: false },
    browser: { enabled: true }
  },
  quietHours: {
    enabled: true,
    start: '22:00',
    end: '08:00'
  }
});

// Export settings
const exportData = await api.settings.export();
```

### Python

```python
from oneclick_sdk import OneClickAPI

api = OneClickAPI(api_key='your-api-key')

# Get settings
settings = api.settings.get_user()

# Update user preferences
api.settings.update_user(
    default_sol_amount=0.2,
    priority_fee=7500,
    auto_launch=True
)

# Create custom preset
preset = api.settings.create_preset(
    name='Conservative Strategy',
    type='LAUNCH_STRATEGY',
    settings={
        'sol_amount': 0.05,
        'slippage_tolerance': 0.02,
        'auto_sell_rules': [
            {
                'type': 'TAKE_PROFIT',
                'trigger': 2.0,
                'action': 50
            }
        ]
    }
)
```
