# Twitter API

The Twitter API enables programmatic management of account subscriptions, feed filtering, and notification settings within OneClick's CT Tracker system.

<Card icon="twitter" title="Twitter Integration">
**Comprehensive Twitter management** through RESTful endpoints:
- Subscribe to and track Twitter accounts
- Configure custom filters and alerts
- Manage notification preferences
- Access real-time tweet feeds
</Card>

## Base URL

```
https://api.oneclick.com/api/twitter
```

All endpoints require authentication via Bearer token in the Authorization header.

---

## Account Management

### Get Tracked Accounts

<Card icon="users" title="GET /accounts">
**Retrieve all tracked Twitter accounts for the authenticated user**
</Card>

**Query Parameters:**
- `is_tracked` - Filter by tracking status: `true`, `false`
- `priority` - Filter by priority level: `high`, `medium`, `low`
- `launch_enabled` - Filter by auto-launch status: `true`, `false`
- `limit` - Results per page (default: 50, max: 200)
- `offset` - Pagination offset

## SDK Examples

### JavaScript/TypeScript

```typescript
import { OneClickAPI } from '@oneclick/sdk';

const api = new OneClickAPI({ apiKey: 'your-api-key' });

// Subscribe to account
const subscription = await api.twitter.subscribe({
  username: 'elonmusk',
  priority: 'high',
  enableLaunch: true,
  maxSolPerLaunch: 0.1
});

// Get feed
const feed = await api.twitter.getFeed({
  filter: 'launched',
  limit: 20
});

// Create filter
const filter = await api.twitter.createFilter({
  name: 'High Confidence Signals',
  conditions: {
    keywords: ['gem', 'alpha'],
    minFollowers: 10000,
    sentiment: 'POSITIVE'
  },
  actions: {
    highlightColor: '#FFD700',
    sendNotification: true
  }
});

// Set up webhook
await api.twitter.setWebhook({
  url: 'https://yourserver.com/webhook',
  events: ['new_tweet', 'filter_match']
});
```

### Python

```python
from oneclick_sdk import OneClickAPI

api = OneClickAPI(api_key='your-api-key')

# Subscribe to account
subscription = api.twitter.subscribe(
    username='elonmusk',
    priority='high',
    enable_launch=True,
    max_sol_per_launch=0.1
)

# Get tracked accounts
accounts = api.twitter.get_accounts(is_tracked=True)

# Create filter
filter_obj = api.twitter.create_filter(
    name='Moonshot Detector',
    conditions={
        'keywords': ['moon', '100x'],
        'min_followers': 5000,
        'sentiment': 'POSITIVE'
    },
    actions={
        'highlight_color': '#00FF00',
        'send_notification': True
    }
)
```
